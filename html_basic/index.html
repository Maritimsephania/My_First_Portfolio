What is HTML?

HTML stands for HyperText Markup Language. It is the standard markup language for creating web pages. HTML describes the structure of a web page using elements and tags. Here are some important things to learn.

Basic Structure

Every HTML document starts with a doctype declaration, followed by the <html> element, which contains the <head> and <body> elements.
